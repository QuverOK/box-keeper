# Seed тестовых данных

Локальный seed создаёт демо-пользователя с 6 хранилищами для ручного тестирования UI, поиска, стопок, layout и граничных размеров.

## Запуск

```bash
cd backend
npm run prisma:seed
```

Требуется настроенный `DATABASE_URL` в `backend/.env` и применённые миграции (`npm run prisma:migrate`).

Повторный запуск безопасен: пользователь с тем же email удаляется и создаётся заново (cascade).

## Учётные данные

| Поле     | Значение по умолчанию  |
| -------- | ---------------------- |
| Email    | `test@boxkeeper.local` |
| Password | `Test1234!`            |
| Name     | `Demo User`            |

Переопределение через env:

- `SEED_USER_EMAIL`
- `SEED_USER_PASSWORD`
- `SEED_USER_NAME`

## Хранилища

| Название                      | Размер комнаты (м) | Сценарий                                      |
| ----------------------------- | ------------------ | --------------------------------------------- |
| Демо: поиск и UI              | 6 × 5 × 2.5        | Поиск, стопки ×2/×3, QR, много вещей, край    |
| Стопка ×50                    | 6 × 6 × 12         | 50 одинаковых коробок в одной колонне         |
| Мин. хранилище                | 0.1 × 0.1 × 0.1    | Коробка 1×1×1 см                              |
| Макс. хранилище               | 30 × 30 × 5        | Коробка на весь объём комнаты                 |
| Layout: перегородки и метки   | 8 × 6 × 3          | Partitions, layout labels, 4 коробки          |
| Неразмещённые коробки         | 4 × 4 × 2.5        | 3 коробки без координат + 1 размещённая       |

## Поиск (хранилище «Демо: поиск и UI»)

| Запрос        | Результат                              |
| ------------- | -------------------------------------- |
| `alpha`       | коробка Alpha-коробка                  |
| `отвёртка`    | вещь в Beta-инструменты                |
| `инструменты` | категория в Beta-инструменты           |
| `архивный`    | описание в Gamma-документы             |
| `2024`        | описание в Gamma-документы             |
